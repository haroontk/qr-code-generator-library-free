# Spec notes — ISO/IEC 18004:2024

Notes on the QR Code Standard ISO/IEC 18004:2024 summarised as I read it. The goal is to:
1. understand the model before materialising into code
2. have something to refer back to without re-reading the whole standard

These notes are my own interpretation of the standard, and does not replace the standard in any form.

## Use of AI

Nothing in this spec will be written by AI. The only ways AI will be used in this project is:
1. to proofread to ensure I haven't typed in anything wrong
2. to suggest improvements or alternative methods, which I can research further
3. to ensure good naming convention and grammar is practiced throughout
